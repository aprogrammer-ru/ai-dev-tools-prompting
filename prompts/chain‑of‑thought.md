## Разработка программы

Разработай алгоритм для поиска самого длинного палиндрома в строке. 
Опиши шаги:
1. Как будешь проверять, является ли подстрока палиндромом?
2. Как эффективно перебирать подстроки?
3. Как отслеживать максимальный найденный палиндром?
4. Затем реализуй решение на Python. 
5. Добавь тесты для случаев: пустая строка, один символ, несколько палиндромов.


## Написание boilerplate

Создай шаблон для модульного теста на Python (unittest) для функции, которая проверяет, является ли число простым. 
Опиши процесс:
1. Какие тестовые случаи нужно покрыть (например, простые/составные числа, крайние значения)?
2. Как структурировать тест-класс?
3. Какие assert-методы использовать?
4. Затем напиши готовый код теста, следуя этим шагам.
