## Генерация кода
Напиши скрипт на Python, который принимает список URL‑адресов и проверяет их доступность (код ответа HTTP). Выведи результат в формате: URL — код статуса — время отклика. Добавь обработку ошибок для некорректных URL и таймаут 5 сек.

## Рефакторинг
Выполни рефакторинг кода, сделав его более читаемым и эффективным. Убери дублирование, добавь понятные имена переменных и комментарии. Язык — JavaScript.

## Boilerplate
Сгенерируй базовый шаблон React‑компонента (функционального) с использованием hooks (useState, useEffect). Компонент должен отображать список элементов из пропса items, иметь состояние для выбранного элемента и обработчик клика. Добавь PropTypes.
